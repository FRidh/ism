ism._ism.is_shadowed()¶

Test whether the receiver is shadowed by any of the walls.

We need to test whether the source can see the receiver. If any of the walls doesn’t block line of sight, and if the receiver is in the field angle, then the receiver is not shadowed and thus the source is effective.

So we need to test using a field angle.

And we need to test whether no objects are in between.

ism.ism.amount_of_sources(order, walls)¶

The amount of potential sources \(N\) up to a certain order \(o\) for a given amount of walls \(w\).

Parameters:
  • order – Order threshold \(o\).
  • walls – Amount of walls \(w\).
Return type:

int

The amount of potential sources \(N\) is given by

\[N = 1 + \sum w \left( w-1 \right)^{o-1}\]
